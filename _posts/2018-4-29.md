## 冒泡排序

冒泡排序比较任何两个相邻的项，如果第一项比第二项大，则交换它们。元素项向上移动至正确的顺序，就好像气泡冒泡至表面一样。

```javascript
function bubbleSort (arr) {
    for(let i=0;i<arr.length;i++){
        for(let j=0;j<arr.length-1-;j++){
                if(arr[j] > arr[j+1]){
                     let arr1 = arr[j];
                     arr[j] = arr[j+1]
                     arr[j+1] = arr1
                }
            }
        }
    return arr
}
```
